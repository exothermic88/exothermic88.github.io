## 🖥️ Mirror or Share Your Screen (GNOME)

### Wireless: Cast via Miracast

- [ ] Install it: `sudo pacman -S gnome-network-displays`
- [ ] Open App Library — `Super+A`
- [ ] Search and open **Gnome Network Displays**
- [ ] Select the display → **Connect**
- [ ] To stop: **Disconnect** or close Network Displays
