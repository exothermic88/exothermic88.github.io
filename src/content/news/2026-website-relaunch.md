Welcome to the new NCOS website!

The old landing page has been rebuilt from the ground up. What changed:

- **Direct downloads** — the Download button now starts the ISO download immediately.
  No more Google Drive, no redirect pages.
- **Live release info** — the download section shows the current version and latest
  changelog, pulled straight from GitHub releases.
- **News feed** — release notes and announcements now live right here on the site.
- **Same wiki, new address** — the [NCOS Wiki](/wiki/home/) moved under `/wiki/`;
  old links redirect automatically.

The site is fully static, built with [Astro](https://astro.build), and hosted on
GitHub Pages. As always, the source is on
[GitHub](https://github.com/exothermic88/exothermic88.github.io) — issues and pull
requests welcome.
