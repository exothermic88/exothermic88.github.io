Hotfix 0.5.1 is out. Install or update the `ncos-hotfix` package to get it:

```bash
sudo pacman -Syu ncos-hotfix
```

## Features

- **Custom Brave browser extension** — effortlessly pull a specific tab out into a new
  window with `Ctrl+Shift+X`. To enable it, open `brave://extensions/`, turn on
  **Developer Mode**, click **Load unpacked**, and select the `BraveExtension` folder
  in `~/.config/`.

## Fixes

- **Discord streaming** — resolved an issue where Discord streaming failed to
  initialize properly.

> For now, run `xwaylandvideobridge` before streaming — autostart for it is coming in a
> future update.

See the [system maintenance guide](/wiki/update/) for how NCOS updates work.
